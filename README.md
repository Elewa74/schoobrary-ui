# Schoobrary — Digital School Library UI

واجهة المكتبة الرقمية لمنصة **سكوبراري (Schoobrary)** — تصميم أمامي ثنائي اللغة (عربي RTL / إنجليزي LTR)، HTML/CSS/JS خالص بدون أي framework.
Bilingual front‑end for the Schoobrary digital school library.

## الصفحات · Pages
| الملف | الوصف |
|------|-------|
| `index.html` | تحويل تلقائي إلى الصفحة الرئيسية · redirects to `search.html` |
| `search.html` | **الصفحة الرئيسية** — ٤ تقسيمات: البحث العام · مكتبة الكتب الرقمية · مكتبة العناصر الرقمية · مكتبة الاختبارات |
| `test-library.html` | **بنك الاختبارات** (صفحة داخلية): بحث + فلاتر مصنّفة + معاينة أسئلة + حل تجريبي + إسناد للطلاب + تصدير PDF |
| `libraries.html` | تحويل إلى `search.html` (لأي رابط قديم) |

## المكوّنات · Sections
- **البحث العام**: بحث حر عن كل موارد التعلّم.
- **مكتبة الكتب الرقمية**: ٦ مكتبات إثرائية — Ask Zad · دليل الكتاب العربي · مكتبة الصحافة والتلفزيون · العربية للحياة · Kalemon K12 · Kalemon Adults.
- **مكتبة العناصر الرقمية**: البحث بالمنهج · البحث بالموضوع · خطط الدروس (نتائج داخل الصفحة بفلاتر وكروت وتابات).
- **مكتبة الاختبارات**: تنتقل إلى بنك الاختبارات (`test-library.html`).

## المميزات · Features
- ثنائية اللغة كاملة (عربي/إنجليزي) مع تبديل فوري RTL/LTR.
- تجربة صفحة واحدة مع انهيار (collapse) عند البحث + زر «العودة».
- بنك اختبارات كامل: فلاتر بعدّادات · معاينة الأسئلة وإظهار الإجابات · حل تجريبي مصحَّح آليًا · إسناد لفصل + تاريخ · تصدير PDF مع مفتاح الإجابات · المفضّلة والمستخدَم مؤخرًا (localStorage).
- تصميم متجاوب (موبايل/تابلت/ديسكتوب) + إتاحة وصول (keyboard/aria).
- هوية موحّدة: خطوط IBM Plex Sans Arabic + Tajawal، وتوكنز ألوان سكوبراري.
